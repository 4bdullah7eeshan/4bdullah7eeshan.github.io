## Learning Resources

Aesthetic Coding is done through various libraries and frameworks. I have classified the learning resources based on the tool that is used.

### p5.js


The following learning resources are centred around [[p5js|p5.js]].

#### Books

> The books are in order of my preference!

- *Aesthetic Programming: A Handbook of Software Studies* by [Winnie Soon](winnie-soon.md), [Geoff Cox](geoff-cox.md).
- *Getting Started with p5.js* by [Lauren-McCarthy](lauren-mccarthy.md), [Casey Reas](casey-reas.md), and [Ben Fry](ben-fry.md). Illustrations by [Taeyoon Choi](taeyoon-choi.md).
- *Generative Design* by [Benedikt Gross](benedikt-gross.md), [Hartmut Bohnacker](hartmut-bohnacker.md), [Julia Laub](julia-laub.md) and [Claudius Lazzeroni](claudius-lazzeroni.md).
- *Learn JavaScript with p5.js* by [Engin Arslan](engin-arslan.md).
